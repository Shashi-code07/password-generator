🔐 Password Generator

A simple and responsive Password Generator built with HTML, CSS, and JavaScript. Generate strong, secure, and customizable passwords instantly.

✨ Features

  • 🔐 Generate strong and secure passwords
  • 📏 Choose password length
  • 🔠 Include uppercase letters
  • 🔡 Include lowercase letters
  • 🔢 Include numbers
  • 🔣 Include special characters
  • 📋 One-click copy to clipboard
  • 📱 Responsive design

🛠️ Technologies Used

  • HTML5
  • CSS3
  • JavaScript (ES6)

🚀 How to Run

  1. Clone the repository
  2. Open index.html in your browser
